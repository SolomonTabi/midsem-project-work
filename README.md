# mid-sem

This repository contains a simple C++ file `project.cpp` that prints "Hello World!".

How to compile locally (Windows):

- With g++:
```powershell
g++ project.cpp -o project.exe
.\project.exe
```

- With Visual Studio `cl.exe` (Developer Command Prompt):
```bat
cl.exe /EHsc project.cpp /Fe:project.exe
.\project.exe
```

To push this project to GitHub, create an empty repository on GitHub and follow the instructions in `push_instructions.txt`.
